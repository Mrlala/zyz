## 1. Demo 简介

**是什么**：中译中是一个"把看不懂的中文翻成人话"的在线词典与翻译工具（H5 网页应用，手机/PC 浏览器均可访问）。

**面向谁**：被"互联网黑话""职场术语""网络热梗"困扰的打工人、新人、跨行业转岗者——看文档像猜谜、开会听不懂行话、刷帖跟不上梗。

**核心功能**：
1. **黑话翻译**：输入"赋能""闭环""抓手"等术语，AI 实时翻译成人话，含释义、例句、风险等级提示
2. **词库浏览**：按分类（职场黑话/互联网梗/废话文学等）浏览词条，支持搜索、收藏
3. **热词排行**：每日热词推荐，投票"能看懂/看不懂"，形成热度榜
4. **用户体系**：注册登录、个人资料、收藏夹、翻译历史、词条提交与审核

## 2. Demo 创作思路

**灵感来源**：工作中遇到"我们要形成闭环，找到抓手，赋能业务"这种话，听十遍也不知道要做什么。网上的"黑话翻译器"多是搞笑段子，没有一个正经的工具能查、能学、能持续积累。

**想解决的问题**：
- 打工人看文档/开会听不懂黑话，不敢问、怕露怯
- 新人入职面对一堆术语，没有系统性的词典可查
- 网络梗更新快，传统词典跟不上，需要一个社区共建、持续更新的词库

**为什么做这个方向**：
- 词典类工具价值明确、边界清晰，适合作为 Demo 快速落地
- "中译中"这个名字自带传播性，容易让人记住
- 后续可扩展性强（多行业术语包、浏览器插件、企业内训版）

## 4. TRAE 实践过程

### 开发流程

整个项目使用 TRAE IDE 完成开发，采用"文档驱动"工作流：需求清单 → 详细设计 → 视觉规范 → 编码实现。

**阶段一：需求与设计**
- 用 TRAE 生成需求规格说明（SRS），编号管理（SRS-001 ~ SRS-XXX）
- 定义技术栈：uni-app + Vue 3 + Pinia + FastAPI + SQLite + DeepSeek AI

**阶段二：前端开发**
- ChatUI 风格主界面（翻译对话流 + 侧边栏抽屉）
- 词库三页（列表/搜索/分类）统一重构，提取 SortBar/EmptyState 共享组件
- 登录注册 + 资料设置（头像选择 + 昵称）+ 路由守卫
- 收藏/历史/提交/审核 完整用户体系

**阶段三：后端开发**
- FastAPI + SQLAlchemy + JWT 鉴权
- DeepSeek AI 翻译接口（含本地词库降级方案）
- 后台管理系统（三角色权限、词库管理、审核、统计）

**阶段四：体验打磨**
- 首页空状态热词推荐
- WordCard featured 变体（渐变色条 + 热度徽章）
- 启动脚本、Docker 部署配置

### 开发心得

- **文档驱动开发**最有效：先写需求清单和设计文档，再编码，减少返工
- **共享组件抽取**是提升效率的关键：SortBar、EmptyState、WordCard 统一了三页样式
- **AI 翻译的降级策略**很重要：无 API Key 时自动降级到本地词库匹配，保证基础可用
